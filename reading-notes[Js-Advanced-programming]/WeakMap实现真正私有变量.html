# WeakMap实现真正私有变量

# WeakMap简介

Es6中新增的”弱映射“是一种新的集合类型，”弱“描述的是Javascript中垃圾回收程序对待WeakMap中键【不算做一次引用，键值对使用完后，会被自动回收】的方式。

WeakMap中的键只能是Object或者继承自Object类型，值的类型没有限制。

基本API请看下面的代码：

// 1、创建几个key(必须是对象类型)
const key1 = {id: 1}
const key2 = {id: 2}
const key3 = {id: 3}

// 2、使用嵌套数组（可迭代对象）初始化一个弱映射
const wm1 = new WeakMap([
    [key1, 'val1'],
    [key2, 'val2'],
])

// 3、set添加
wm1.set(key3, 'val3')
// 4、get按键查询值
console.log(wm1.get(key3))  // val3
// 5、还有delete和has
wm1.delete(key1)
console.log(wm1.has(key1))  // false

# 实现私有变量

实现方法： 私有变量会存储在弱映射中，以对象实例为键，以私有成员的字典为值，如：[实例，{属性名：属性值}]

# 代码如下：

const wm = new WeakMap()

class User {
    constructor(prop, id) {
        this.setPrivate(prop, id)
    }

    // 添加私有变量
    setPrivate(prop, val) {
        // 将所有私有成员保存在一个对象中，初始时为空
        const privateMembers = {} || wm.get(this)   // 以本实例为键
        // 添加到privateMembers
        privateMembers[prop] = val                  // 使用变量，必须用[]语法
        // 添加到wm中
        wm.set(this, privateMembers)
    }

    // 获取私有变量
    getPrivate(prop) {
        // 先获取privateMembers对象，再获取里面的值
        return wm.get(this)[prop]
    }
}

// new一个User实例user1,同时创建一个私有变量idProp=123
const user1= new User('idProp', 123)
// 只能通过向外暴露的函数来访问该变量，因此是私有变量
console.log(user1.getPrivate('idProp'))			// 123

# 如何解决呢？

这就要用到闭包了：

// 闭包
const User = (() => {
    const wm = new WeakMap()

    class User {
        constructor(prop, id) {
            this.setPrivate(prop, id)
        }

        // 添加私有变量
        setPrivate(prop, val) {
            // 将所有私有成员保存在一个对象中，初始时为空
            const privateMembers = {} || wm.get(this)   // 以本实例为键
            // 添加到privateMembers
            privateMembers[prop] = val                  // 使用变量，必须用[]语法
            // 添加到wm中
            wm.set(this, privateMembers)
        }

        // 获取私有变量
        getPrivate(prop) {
            // 先获取privateMembers对象，再获取里面的值
            return wm.get(this)[prop]
        }
    }
    return User
})()

// new一个User实例user1,同时创建一个私有变量idProp=123
const user1= new User('idProp', 123)
// 只能通过向外暴露的函数来访问该变量，是私有变量
console.log(user1.getPrivate('idProp'))
